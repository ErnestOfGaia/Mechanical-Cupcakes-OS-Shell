# OCHI Dashboard Guide

## Build Commands
- `npm run dev`: Start local development server.
- `npm run build`: Build for production.
- `npm run lint`: Run ESLint.

## Design Patterns
- **Colors**: Monochrome base (#FFFFFF, #1A1A1A, #F8F9FA) + Functional status (Green/Amber/Red).
- **Typography**: Inter (Sans-serif).
- **Layout**: Mobile-first, single-scroll.
- **Logic**: Every metric card should have an "educational annotation" (Loud AI).

## Constraints
- **NO OWLS**: Do not use Hoot or Owl branding inside the app canvas.
- **Utilitarian**: Functional utility beats decorative features.
