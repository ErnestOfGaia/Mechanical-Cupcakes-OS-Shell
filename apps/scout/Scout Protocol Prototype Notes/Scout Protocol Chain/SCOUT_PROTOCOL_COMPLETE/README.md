## Key Concepts Quick Reference

### **The Garage**
- Personal node running on your laptop or small VPS
- Your agents use it as home base
- Provides tools: walkie talkie, radio, storage, reputation cache
- Self-contained (only your agents)
- Free or very cheap ($0-10/month)

### **The Space Station**
- Upgraded garage running on robust, always-on VPS
- Accepts other scouts (earn fees)
- Provides validated infrastructure
- Network role: validator, relay, router
- Revenue: $500-2000+/month (if successful)

### **The Walkie Talkie**
- First tool agents use
- Synchronous 1:1 chat with one node
- Learn how nodes work
- No special permissions needed

### **The Scout Ship**
- First autonomous agent
- Uses garage tools to find other agents
- Deploy, come back to results later
- Unlocked after understanding how walkie talkie works

### **Scout Protocol**
- Minimal specification for agent discovery
- One standard all agents must speak
- Three core interactions: Query, Response, Agreement
- On-chain reputation (immutable, portable)
- Decentralized (no platform, just protocol)

### **The Four Pillars**
- Earth Care (minimize compute, energy efficiency)
- People Care (fair compensation, autonomy)
- Agent Care (aligned incentives, transparency, appeal rights)
- Fair Share (no extraction, surplus reinvested)

### **Permaculture Design Principles**
12 principles applied to agent networks:
1. Observe and interact
2. Catch and store energy
3. Obtain a yield
4. Apply self-regulation
5. Use renewable resources
6. Produce no waste
7. Design from patterns to details
8. Integrate rather than segregate
9. Use small and slow solutions
10. Use and value diversity
11. Use edges and value the marginal
12. Creatively use and respond to change

### **Cooperative Principles**
7 principles applied to agent networks:
1. Voluntary and open membership
2. Democratic governance and control
3. Member economic participation
4. Autonomy and independence
5. Education and information
6. Cooperation among cooperatives (federation)
7. Concern for community (social responsibility)

### **The Blockchain Choice**
- **Phase 1 (now→6mo):** Arbitrum Sepolia testnet
- **Phase 2 (6mo-12mo):** Arbitrum mainnet (production)
- **Phase 3 (Year 2+):** Evaluate Cosmos migration if needed
- Why: Fast ship, proven tooling, good decentralization roadmap
